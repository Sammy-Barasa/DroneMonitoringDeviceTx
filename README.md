# DroneMonitoringDeviceTx


## Drone Monitoring Device Transmitter 

Drone  monitoring device transmitter contains a micro controller (arduino mega), sensors that capture salient realtime drone data as raw, pitch, yaw, latitude, longitude and height above ground. These obtained data are captured and transmitted via longrange communication (LoRa module) to a receiving module that then sends the data over the interenet.
